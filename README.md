# Place/Pixel Thing

Based on https://github.com/xSke/Pxls

Just messing around in node.js

Installation instructions:
## 1 - Clone it
```git clone https://github.com/liamcottam/place```
## 2 - Install node.js libs
```npm install```
## 3 - Generate data file
```node create-datafile.js```
## 4 - Run it
```node app.js```
